# 令和ボイス (Reiwa Voice) - 匿名陳情システム 公式リポジトリ

本リポジトリは、日本中の国民から陳情を集め、匿名で当該自治体の議会へ送付するシステム「令和ボイス」の公式ホームページのソースコードです。
「行政のオープンソース化（シビックテック）」というビジョンを自ら体現するため、本サイトのソースコードを公開しています。

* **発案:** へずまりゅう（奈良市議会議員）
* **システム開発:** 丸吉 孝文（シビックテックエンジニア）

### 理念：なぜ行政をオープンソース化するのか (Philosophy & Origins)
私のシビックテック（市民参加型DX）の原点は、2016年に遡ります。
熊本地震の直後に開催されたハッカソン「Startup Weekend ALL for Kumamoto」において、テクノロジーを用いた災害支援・社会課題解決（Tech for Good）のプロダクトを実装し、優勝を果たしました。

この経験から、**「コードとデータには、社会のレジリエンス（回復力）を高め、人々の痛みを直接的に解決する力がある」**と確信しました。
昨日今日思いついたDXではありません。10年前から一貫して「社会課題の解決」にエンジニアリングを捧げてきた実務家として、高額な税金に依存する古い行政システムをハックし、本当に機能する「行政のオープンソース化」を推進しています。
